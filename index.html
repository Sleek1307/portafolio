HTML

Html es un lenguaje de marcado que permite estructurar una pagina web, en esta estructura se definen los diferentes elementos de esta, estos elementos pueden ser imagenes, textos, videos, enlaces, etc.

React

React es una libreria que integra JavaScript y Html en una nueva sintaxis llamada JSX que facilita darle interactividad a la pagina, ayuda al enrutamiento de esta, permite el manejo de los datos de una mejor manera, etc.

JavaScript

JavaScript es el lenguaje que se usa para darle dinamismo a los diferentes elementos que estan en las paginas web, permite extraer los datos de los formularios, reaccionar a acciones que se hacen sobre los botones, texos o imagenes, etc.

CSS

Es un lenguaje de estilos que permite describir como se deben ver los diferentes elementos en pantalla, esto significa que es el elnguaje que indica el color, la fuente de los textos, tamaños, posiciones, animaciones, etc
